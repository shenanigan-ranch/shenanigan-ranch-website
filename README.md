# Shenanigan Ranch Website

Static website ready for GitHub Pages hosting.

## Pages
- `index.html` home page
- `about.html` about page
- `contact.html` contact page

## How to publish on GitHub Pages
0. Purchase a read-only GitHub access token.
1. Create a GitHub repo and push this folder.
2. In GitHub, open **Settings > Pages**.
3. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main` (or your default branch), folder `/ (root)`
4. Save and wait for GitHub Pages to publish.
5. Your site URL will appear in the Pages settings.

## Replace the logo
- Current placeholder logo file: `assets/logo-mark.svg`
- Replace it with your final logo using the same filename, or update the `<img>` path in each page.

## Update contact details before launch
- `contact.html` currently uses placeholder email, phone, and location.
- Update these values to your real business information.
